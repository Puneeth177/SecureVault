SecureVault Authentication Debug

This page helps debug authentication issues. Use it to test registration and login with proper validation feedback.

Password Requirements:

  • Minimum 8 characters long
  • At least one uppercase letter (A-Z)
  • At least one lowercase letter (a-z)
  • At least one number (0-9)
  • At least one special character (@$!%*?&)
  • Cannot contain common weak patterns like "password123", "admin123", etc.

Example strong password: MyPass123!
